# Leap! Pad

A Minecraft mod for Fabric 1.20.1 that lets players travel between friend-hosted worlds through prismarine portals.

Build a portal frame from prismarine blocks, throw a written book containing a connection address into the portal to link it, then walk through to travel.

Includes a character profile system — maintain separate characters with their own inventories across different worlds.

---

## Mod Family

| Mod | Purpose |
|---|---|
| **Leap! Pad** (this mod) | Portal travel and profile system |
| **Leap! Forward** | Loads the destination world's mod environment on the client |
| **Leap! Backwards** | Keeps the host's home world alive while the host travels |

Leap! Forward and Leap! Backwards both require Leap! Pad. They are independent of each other.

---

## Requirements

- Minecraft 1.20.1
- Fabric Loader 0.19.2 or newer
- Fabric API 0.92.9 or newer
- Java 17
